Import comments from Facebook comments box into Wordpress database.

== Description ==

FB Comments Box Importer is a simple plugin that lets you import comments from your Facebook comments box directly into a Wordpress comments system.
Like our other plugin (FB Comments Importer) it gives your site an SEO boost and it makes it look even more alive.

If you need to import comments from your facebook page then check out our other plugin - FB Comments Importer.

For any issues with the plugin, please open a support ticket and we will fix it ASAP!

== Installation ==

1. Unzip `fb-comments-box-importer.zip` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a Facebook APP at developers.facebook.com/apps/
4. Configure the plugin by going to the `FB Comments Box Importer` page
